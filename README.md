Pattern will help you solve almost any pattern problem in interviews.


Every pattern = combination of:

Spaces + Characters (stars/numbers) + Formula
Types of Patterns

All questions fall into these 5 types:

1. Square / Rectangle
2. Right Triangle
3. Pyramid
4. Inverted Pyramid
5. Diamond

6. Whenever you see a pattern:

Step 1: Count rows → i
Step 2: Break into:
spaces
characters
Step 3: Identify pattern:
Pattern Type	Formula
Increasing	i
Decreasing	n-i
Pyramid	2*i+1
Inverted Pyramid	2*(n-i)-1
